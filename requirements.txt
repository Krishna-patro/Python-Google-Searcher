# Python Google Searcher

This project demonstrates how to programmatically perform a Google search and retrieve the resulting URLs using the **`googlesearch-python`** library. It's a great starting point for building simple search bots or data-gathering tools.

## Features
- Takes a search query as input.
- Retrieves a specified number of search result URLs.

## Installation
1. Clone this repository.
2. Install the required library:
   ```
   pip install -r requirements.txt
   ```

## Usage
1. Open the `Google Search.py` file.
2. Change the `search_term` variable to your desired query.
3. Run the script from your terminal:
   ```
   python google_search.py
   ```
